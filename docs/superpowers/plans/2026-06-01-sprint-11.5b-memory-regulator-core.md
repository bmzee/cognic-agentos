> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the **regulator-core** half of Sprint 11.5b on top of the merged 11.5a substrate — the erasure trio (`forget` / `redact` / regulator-erasure), the tombstone reaper, the real Redis `memory.write_freeze` kill-switch (replacing the 11.5a fail-loud seam), the scratch-Postgres fallback, and the core RBAC additions — per ADR-019. Surfaces (`export`, `learning_surface` validator, portal `/memory` API, UI emit-hooks, vector→real-recall wiring) are **11.5c**, NOT here.

**Architecture:** The 11.5a `MemoryAPI` / `MemoryGate` / `MemoryAdapter` substrate is unchanged in shape; 11.5b *extends* it. Erasure is chain-linked exactly like a write: `forget` / `redact` / regulator-erasure each mutate `memory_records` **inside** a `DecisionHistoryStore.append_with_precondition` transaction that atomically emits a write-side `memory.forget` / `memory.redact` / `memory.regulator_erasure` event (digest/metadata only — never the value). The kill-switch is a drop-in conformer to the existing `MemoryKillSwitchInterrogator` seam (zero gate-code change). The reaper is an off-gate Doctrine-F asyncio loop over an on-gate purge primitive. Scratch durability-on-Redis-loss is a routing adapter, not a loosening of `PostgresMemoryAdapter.put()`.

**The six locked design decisions (baked into the tasks; a worker MUST NOT paper over them):**
1. **`RedactionSpan` = JSON field-path only** — `RedactionSpan(path: tuple[str, ...], replacement: object = "[REDACTED]")`. A missing/non-container path refuses with the typed `memory_redaction_path_invalid`. NO regex / byte-span masking.
2. **Kill-switch Redis value shape** — `cognic:killswitch:memory_write_freeze:{tenant_id}` → JSON carrying at least `frozen: bool` / `updated_at` / `actor_id` / `reason`. `is_write_frozen()` uses cached last-known-good while cache age ≤ `memory_kill_switch_cache_ttl_s` (≤ 60); Redis unreachable + stale/absent cache ⇒ **fail-closed frozen (True)**.
3. **`EmergencyRBACScope` is its own family** (NOT folded into `MemoryRBACScope`); ships `EMERGENCY_SCOPES` + `Actor`/`RequireScope` widening + namespace-disjoint tests.
4. **Regulator-erasure authz = command-metadata + chain-of-custody in core; RBAC enforcement deferred to the 11.5c portal.** Core `MemoryAPI` has no `Actor`/scope set — it does NOT pretend to do `RequireScope`. `forget(reason="regulator_erasure")` requires a `RegulatorErasureCommand(regulator_order_id, requester_scope, subject_id)` with `requester_scope == "memory.regulator_erasure"`; missing/wrong ⇒ `memory_regulator_erasure_metadata_required`. `actor_id` comes from the bound context. The fields land on the `memory.regulator_erasure` chain payload.
5. **Scratch-PG fallback fires ONLY on Redis-unavailable**, not arbitrary Redis write bugs — via a `RoutingMemoryAdapter`, NOT by loosening `PostgresMemoryAdapter.put()`. Scratch reads try Redis then consult PG fallback rows. **Expired `retention_until` rows are a miss immediately** (the T4/T7 retention-expiry invariant), before the reaper physically purges them.
6. **Closed-enum lifecycle refusal reasons** added to `MemoryRefusalReason` (count-pinned like 11.5a T1/T9): `memory_record_not_found`, `memory_record_already_tombstoned`, `memory_redaction_path_invalid`, `memory_regulator_erasure_metadata_required`.

class _FakeRedis:
    """Duck-typed get/set; `available=False` simulates an unreachable backend."""
    def __init__(self, available=True):
        self.store, self.available = {}, available

        if not self.available:

from __future__ import annotations

import json
from collections.abc import Callable
from datetime import UTC, datetime
from typing import Any, Protocol, runtime_checkable

_KEY_PREFIX = "cognic:killswitch:memory_write_freeze:"


def _write_freeze_key(tenant_id: str) -> str:
    return f"{_KEY_PREFIX}{tenant_id}"


@runtime_checkable
class _AsyncRedisKVLike(Protocol):
    async def get(self, key: str) -> Any: ...
    async def set(self, key: str, value: Any, **kwargs: Any) -> Any: ...


def _utcnow() -> datetime:
    return datetime.now(UTC)


class RedisMemoryWriteFreezeKillSwitch:
    """Per-tenant memory.write-freeze probe with fail-closed cached grace.

    is_write_frozen(): read Redis → parse {frozen, updated_at, actor_id, reason}
    → refresh the per-tenant last-known-good cache → return frozen. On a Redis
    error: serve the cached value while its age <= cache_ttl_s; otherwise (stale
    or no cache) FAIL CLOSED (return True). A malformed value also fails closed.
    """

    def __init__(
        self,
        *,
        redis_client: _AsyncRedisKVLike,
        cache_ttl_s: int,
        clock: Callable[[], datetime] = _utcnow,
    ) -> None:
        self._redis = redis_client
        self._cache_ttl_s = cache_ttl_s
        self._clock = clock
        self._cache: dict[str, tuple[bool, datetime]] = {}   # tenant_id -> (frozen, observed_at)

    async def is_write_frozen(self, *, tenant_id: str) -> bool:
        try:
            raw = await self._redis.get(_write_freeze_key(tenant_id))
        except Exception:
            # Redis unreachable: cached last-known-good while fresh, else fail-closed.
            cached = self._cache.get(tenant_id)
            if cached is not None:
                frozen, observed_at = cached
                if (self._clock() - observed_at).total_seconds() <= self._cache_ttl_s:
                    return frozen
            return True
        if raw is None:
            self._cache[tenant_id] = (False, self._clock())     # absent key => not frozen
            return False
        try:
            doc = json.loads(raw if isinstance(raw, str) else raw.decode())
            frozen = doc["frozen"]
            _custody = (doc["updated_at"], doc["actor_id"], doc["reason"])   # ALL required
            if not isinstance(frozen, bool):                    # non-bool `frozen` is malformed —
                raise ValueError("frozen must be a JSON bool")  # do NOT bool()-coerce (0 => fail-open)
        except (ValueError, TypeError, KeyError, AttributeError):
            return True                                         # malformed / partial => fail-closed
        self._cache[tenant_id] = (frozen, self._clock())
        return frozen

    async def set_write_freeze(
        self, *, tenant_id: str, frozen: bool, actor_id: str, reason: str
    ) -> None:
        """Ops/portal write surface (the portal RBAC gate is 11.5c). Writes the
        frozen-state JSON; the read path + cache pick it up on the next probe."""
        payload = json.dumps(
            {"frozen": frozen, "updated_at": self._clock().isoformat(),
             "actor_id": actor_id, "reason": reason}
        )
        await self._redis.set(_write_freeze_key(tenant_id), payload)


__all__ = ("RedisMemoryWriteFreezeKillSwitch", "_write_freeze_key")

import copy
def _apply_redaction(value: object, path: tuple[str, ...], replacement: object) -> object:
    """Deep-copy `value` and replace the leaf at `path` with `replacement`.
    Raises ValueError on an empty path, a missing key, or a non-mapping midpoint
    — the caller maps it to memory_redaction_path_invalid (locked: field-path only)."""
    if not path:
        raise ValueError("redaction path must be non-empty")
    out = copy.deepcopy(value)
    cursor = out
    for seg in path[:-1]:
        if not isinstance(cursor, dict) or seg not in cursor:
            raise ValueError(f"redaction path segment {seg!r} not a mapping key")
        cursor = cursor[seg]
    leaf = path[-1]
    if not isinstance(cursor, dict) or leaf not in cursor:
        raise ValueError(f"redaction leaf {leaf!r} absent")
    cursor[leaf] = replacement
    return out

from __future__ import annotations
import asyncio, logging
from typing import TYPE_CHECKING, Protocol
if TYPE_CHECKING:
    from cognic_agentos.core.memory.storage import MemoryAdapter

logger = logging.getLogger(__name__)

class _MemoryReaperSettings(Protocol):
    memory_tombstone_window_s: int
    memory_reaper_interval_s: int

class MemoryTombstoneReaper:
    def __init__(self, *, adapter: MemoryAdapter, settings: _MemoryReaperSettings) -> None:
        self._adapter = adapter
        self._settings = settings

    async def run_once(self) -> int:
        try:
            return await self._adapter.purge_expired(
                tombstone_window_s=self._settings.memory_tombstone_window_s)
        except asyncio.CancelledError:
            raise
        except Exception:
            logger.exception("MemoryTombstoneReaper sweep failed; loop continues to next interval.")
            return 0

    async def run_forever(self) -> None:
        interval = self._settings.memory_reaper_interval_s
        while True:
            await self.run_once()
            try:
                await asyncio.sleep(interval)
            except asyncio.CancelledError:
                logger.info("MemoryTombstoneReaper.run_forever cancelled — clean shutdown")
                raise

__all__ = ["MemoryTombstoneReaper"]
