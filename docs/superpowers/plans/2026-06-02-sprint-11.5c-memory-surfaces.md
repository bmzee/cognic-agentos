# Sprint 11.5c — Agent Memory Governance Surfaces (ADR-019) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (one fresh worker per task; two-stage main-thread review: spec-compliance + trust-but-verify gate-ladder; halt-before-commit per task with explicit commit token by explicit paths). Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the **surfaces** around the 11.5a/11.5b governed-memory substrate — export, the `learning_surface` manifest validator, the portal `/memory` API, the `memory.*` UI emit-hooks, and the vector→real recall wiring — so a bank can inspect, export, redact, forget, and semantically recall governed memory through HTTP + CI surfaces, while the substrate itself stays **DI-tested, not harness-injected** (the honesty marker held since 11.5a).

**Architecture:** Every surface is additive and rides the existing enforcement spine. The portal goes through a per-request `MemoryAPI` (never imports `core.memory.storage` — the arch guard at `tests/unit/architecture/test_memory_layer_c_no_direct_storage.py` forbids it). Export reuses the ADR-016 `ObjectStoreAdapter` retention discipline — the 7-year floor is mirrored into `Settings.memory_export_retention_seconds` (default/floor, configurable longer; no second bundler, no `supply_chain.py` edit) and the bucket into `Settings.memory_export_bucket` (shape-validated to the shipped `local_fs` bucket contract). UI hooks add `memory.*` projector entries to the existing `_DECISION_HISTORY_TYPED_PROJECTORS` dispatch — no new event schema (ADR-020 backward-compat). Vector recall extends `recall_episodes` with an optional `query` and an opt-in `MemoryVectorIndex`; restricted-class records are excluded from indexing (Wave-1 bank-safe default). Value never enters the chain anywhere (the 11.5a/11.5b invariant extends to `memory.export`).

**Tech Stack:** Python 3.12 (`uv run`), FastAPI (portal), Pydantic v2 (DTOs/events), SQLAlchemy async (chain linkage via `DecisionHistoryStore.append_with_precondition` / plain `append`), OPA/Rego (existing gate), qdrant via the bundled `VectorAdapter`, `ObjectStoreAdapter` (`local_fs` driver). No new third-party deps.

**Design forks (resolved 2026-06-02 after grounding in the landed 11.5a/11.5b code):**
- **F1 — Recall query + embedding governance.** `recall_episodes` gains optional `query: str`. `threshold == 0.0` → unchanged long_term+purpose view. `threshold > 0.0` → requires `query` AND a wired `MemoryVectorIndex`; searches the index, then **intersects with the governed, agent-scoped active `long_term` set** (authz-correct via `list_for_subject`); threshold-filters; ranks by similarity. No query / no index → typed `MemoryOperationRefused("memory_vector_recall_unavailable")` (refusal taxonomy **16 → 17**). **Index-on-write** only for `long_term`, only when an index is wired, and **only for records whose `data_classes` are disjoint from `RESTRICTED_DATA_CLASSES`** (Wave-1 bank-safe default — restricted records stay recallable via the 0.0 view, never embedded into qdrant). **Index-on-write is best-effort (a derived index):** the governed record is already durably written + chain-linked when indexing runs; a failed embed is **caught + logged (record_id + purpose + exception class only — never the value/text) and does NOT fail `remember`** (the vector index is a recall accelerator, not the system of record; a missed embed just means that record is not vector-rankable until a future reindex).
- **F2 — Portal is surface-only, agent-scoped.** The 11.5b `forget`/`redact`/`list_for_subject` primitives are `(tenant + agent)`-scoped; there is no get-by-record-id and no tenant+subject read. The portal builds a per-request `MemoryAPI` bound to an **operator context** (`is_subagent=False`; `served_subject` = requested subject; `agent_id` supplied in the request). `GET /memory/records?subject&agent_id` → **metadata only, no values** (incl. owning `agent_id` so the UI drives the POSTs). `POST .../forget|redact` + `POST /export` are `(agent, subject)`-scoped. **Whole-subject cross-agent erasure is NOT in 11.5c** — it needs a tenant+subject `storage.py` path (a CC stop-rule edit) → deferred (valve / 13.5). Add one small **value-free governed** `MemoryAPI.list_records(subject) -> list[MemoryRecordMetadata]` (since `list_for_subject` returns only ids).
- **F3 — Export = retention-discipline only.** Serialize the authorized `(agent, subject)` records to canonical bytes (**values included** — caller holds `memory.export.read`), persist via `ObjectStoreAdapter.put(bucket, key, body, retention_seconds=...)` where **bucket + retention are injected from validated Settings** (`memory_export_retention_seconds` default/floor = the 7-year ADR-016 value, longer allowed + below-floor rejected; `memory_export_bucket` shape-validated to the `local_fs` `_BUCKET_RE`; **no `supply_chain.py` edit** — the floor is an inline mirror in `core/config.py`), object-store-safe structured key (deterministic `(tenant, agent, subject)` identity segment + a **unique per-export id** — re-exports are distinct, NOT overwrite), fail-closed `MemoryExportPersistenceFailed`, `memory.export` chain row carries **digest + key + count + retention only**. **Cosign-signing of the archive is DEFERRED** (the D3 valve candidate).

```python
"""Sprint 11.5c — [learning_surface] manifest validator (ADR-019).

CRITICAL CONTROL (CLI build-time half of the memory-governance trust gate per
ADR-008 + ADR-019). Validates a pack's [learning_surface] declaration: a
closed-enum mode + a learnable-data-class allow-list. Refuses restricted data
classes declared as learnable (a pack cannot self-grant customer_pii /
payment_data / credentials / regulator_communication as a learning surface).
Dual-path lookup ([tool.cognic.learning_surface] canonical per ADR-019 §52 +
[learning_surface] compat alias) per feedback_dual_path_doctrine.
"""

from pathlib import Path
from typing import Any

from cognic_agentos.cli import ValidatorFinding
from cognic_agentos.cli._governance_vocab import (
    DataClass,
    LearningSurfaceMode,
    RESTRICTED_DATA_CLASSES,
)
import typing

_VALID_MODES = frozenset(typing.get_args(LearningSurfaceMode))
_VALID_DATA_CLASSES = frozenset(typing.get_args(DataClass))

# Canonical path is [tool.cognic.learning_surface] (ADR-019 §52); top-level
# [learning_surface] is a compat alias (the broader dual-path convention).
_LOCATIONS: tuple[tuple[str, tuple[str, ...]], ...] = (
    ("tool.cognic.learning_surface", ("tool", "cognic", "learning_surface")),
    ("learning_surface", ("learning_surface",)),
)


def _resolve(data: dict[str, Any], accessor: tuple[str, ...]) -> Any:
    node: Any = data
    for k in accessor:
        if not isinstance(node, dict) or k not in node:
            return None
        node = node[k]
    return node


def _violation(*, prefix: str, failure_mode: str, message: str, **extra: Any) -> ValidatorFinding:
    # Single closed-enum reason per ADR-019 §52; failure_mode discriminates the sub-case
    # (mirrors supply_chain.py's payload.failure_mode pattern).
    return ValidatorFinding(
        severity="refusal",
        reason="learning_surface_violation",
        message=message,
        payload={"block_path": prefix, "failure_mode": failure_mode, **extra},
    )


def validate(data: dict[str, Any], pack_path: Path) -> list[ValidatorFinding]:
    findings: list[ValidatorFinding] = []
    for prefix, accessor in _LOCATIONS:
        block = _resolve(data, accessor)
        if block is None:
            continue
        if not isinstance(block, dict):
            findings.append(
                _violation(
                    prefix=prefix,
                    failure_mode="invalid_shape",
                    message=f"[{prefix}] must be a table; got {type(block).__name__}",
                )
            )
            continue
        mode = block.get("mode")
        if mode is not None and mode not in _VALID_MODES:
            findings.append(
                _violation(
                    prefix=prefix,
                    failure_mode="mode_invalid",
                    message=f"[{prefix}].mode={mode!r} not in {sorted(_VALID_MODES)}",
                    value=mode,
                )
            )
        ldc = block.get("learnable_data_classes")
        if ldc is not None and not isinstance(ldc, list):
            # A bare string must NOT iterate per-character (`"public"` → 6 findings);
            # refuse the shape once and skip the member loop.
            findings.append(
                _violation(
                    prefix=prefix,
                    failure_mode="learnable_data_classes_not_list",
                    message=f"[{prefix}].learnable_data_classes must be an array; got {type(ldc).__name__}",
                )
            )
            ldc = []
        for dc in ldc or []:
            if not isinstance(dc, str) or dc not in _VALID_DATA_CLASSES:
                findings.append(
                    _violation(
                        prefix=prefix,
                        failure_mode="data_class_invalid",
                        message=f"[{prefix}].learnable_data_classes: {dc!r} is not a DataClass",
                        value=dc,
                    )
                )
            elif dc in RESTRICTED_DATA_CLASSES:
                findings.append(
                    _violation(
                        prefix=prefix,
                        failure_mode="data_class_restricted_forbidden",
                        message=(
                            f"[{prefix}].learnable_data_classes: {dc!r} is a restricted class "
                            f"and may not be declared a learning surface"
```

```python
Cosign-signing of the archive is DEFERRED (the D3 valve candidate).

class MemoryExportPersistenceFailed(Exception):
    """Fail-closed wrapper for an object-store persistence failure during export.
    Mirrors SigstoreBundlePersistenceFailed: NO raw archive bytes in the message,
    only the sha256 digest + length + object key + exception class."""


def _export_object_key(*, tenant_id: str, agent_id: str, subject: SubjectRef, export_id: str) -> str:
    # Object-store-safe key (LocalObjectStoreAdapter._KEY_RE = ^[a-z0-9][a-z0-9._/-]{0,255}$
    # — lowercase, NO ':'). tenant_id / agent_id / subject.canonical (human:<id>) can carry
    # ':' / uppercase / '@' / '/', so the identity tuple is hashed to ONE lowercase-hex
    # segment; export_id (uuid4().hex, lowercase hex) keeps re-exports distinct (NOT
    # overwrite). Deterministic per (tenant, agent, subject).
    identity = hashlib.sha256(canonical_bytes([tenant_id, agent_id, subject.canonical])).hexdigest()
    return f"memory-exports/{identity}/{export_id}.json"


def _serialize(hits: list[MemoryHit]) -> bytes:
    # Canonical bytes (core.canonical.canonical_bytes — the same wire-form the chain
    # uses), values INCLUDED (authorized export). Stored values are JSONB-origin
    # (JSON-native: dict/list/str/num/bool/None) so canonical_bytes handles them; a
    # tuple would be rejected by the canonical-form gate but cannot arise from a JSONB
    # read. The archive digest (below) is sha256 over THESE bytes — deterministic.
    body = [
        {
            "record_id": str(h.record_id),
            "tier": h.tier,
            "purpose": h.purpose,
            "data_classes": list(h.data_classes),
            "block_kind": h.block_kind,
            "created_at": h.created_at.isoformat(),
            "value": h.value,
        }
        for h in hits
    ]
    return canonical_bytes(body)


async def export_memory(
    *,
    hits: list[MemoryHit],
    subject: SubjectRef,
    context: MemoryCallerContext,
    object_store: ObjectStoreAdapter,
    audit: DecisionHistoryStore,
    bucket: str,
    retention_seconds: int,
) -> ExportReceipt:
    # bucket + retention_seconds are caller-supplied (MemoryAPI.export reads them
    # from validated Settings); this primitive owns no retention policy + never
    # floors them — the 7-year ADR-016 floor is enforced at Settings construction.
    # Distributed-atomicity boundary (deliberate): PUT-then-append across two
    # transactionless systems; an append failure AFTER a successful PUT leaves the
    # archive orphaned (exists under retention, no chain row) — the CONSERVATIVE
    # failure (over-retained data, never a chain row referencing a missing
    # artifact). No compensating delete (the archive is under a WORM retention
    # window). The append exception propagates.
    archive_bytes = _serialize(hits)
    archive_sha256 = hashlib.sha256(archive_bytes).hexdigest()
    export_id = uuid.uuid4().hex
    key = _export_object_key(
        tenant_id=context.tenant_id, agent_id=context.agent_id, subject=subject, export_id=export_id
    )
    try:
        await object_store.put(
            bucket,
            key,
            archive_bytes,
            retention_seconds=retention_seconds,
        )
    except Exception as exc:
        raise MemoryExportPersistenceFailed(
            f"memory export persistence failed for subject={subject.canonical!r} "
            f"agent_id={context.agent_id!r} key={key!r}: class={type(exc).__name__} "
            f"archive_sha256={archive_sha256} archive_len={len(archive_bytes)}"
        ) from None
    await audit.append(
        DecisionRecord(
            decision_type="memory.export",
            request_id=f"memory-export-{export_id}",
            payload={
                "op": "export",
                "subject_ref": subject.canonical,
                "agent_id": context.agent_id,
                "object_key": key,
                "archive_sha256": archive_sha256,
                "record_count": len(hits),
                "retention_seconds": retention_seconds,
            },
            actor_id=context.actor_id,
            tenant_id=context.tenant_id,
            iso_controls=_MEMORY_EXPORT_ISO_CONTROLS,
        )
    )
    return ExportReceipt(
        object_key=key, archive_sha256=archive_sha256, record_count=len(hits)
    )


__all__ = ("export_memory", "MemoryExportPersistenceFailed")
```

```python
    _forget = RequireScope("memory.forget")
    _regulator = RequireScope("memory.regulator_erasure")

        # Operator context: is_subagent=False (a human/service operator, not a child agent);
        # served_subject = the requested subject; identity from the Actor + request body.

            tenant_id=actor.tenant_id, agent_id=agent_id, actor_id=actor.subject,
            served_subject=subject, is_subagent=False, long_term_writes_allowed=False,
            cross_subject_recall=False,
            memory_read_capabilities=frozenset({"memory_read.task", "memory_read.long_term"}),

        )

    async def list_records(
        actor: Annotated[Actor, Depends(_read)],
        subject_kind: str, subject_id: str, agent_id: str,
    ):

        api = memory_api_factory(_operator_context(actor, agent_id=agent_id, subject=subject))
        try:

        except MemoryOperationRefused as exc:
            raise HTTPException(status_code=409, detail={"reason": exc.reason})
        return [MemoryRecordMetadataResponse.model_validate(m, from_attributes=True) for m in meta]

    @router.post("/records/{record_id}/forget")

                     actor: Annotated[Actor, Depends(_forget)]):

        # `_regulator.__call__` reference enforces NOTHING — do not write that.
        if body.reason == "regulator_erasure":

                raise HTTPException(
                    status_code=403,
                    detail={"reason": "scope_not_held", "required_scope": "memory.regulator_erasure"},
                )

                raise HTTPException(status_code=403, detail={"reason": "actor_type_must_be_human"})

                regulator_order_id=body.erasure_command.regulator_order_id,
                requester_scope=body.erasure_command.requester_scope,
                subject_id=body.erasure_command.subject_id,
            )
            if body.erasure_command is not None
            else None
        )

        except MemoryOperationRefused as exc:
            status = 404 if exc.reason == "memory_record_not_found" else 409
            raise HTTPException(status_code=status, detail={"reason": exc.reason})
        return ForgetReceiptResponse.model_validate(receipt, from_attributes=True)

    # redact analogous (scope memory.redact, no human gate). export carries _human +
    # _export as Depends sub-dependencies (always human + always memory.export.read).
```
